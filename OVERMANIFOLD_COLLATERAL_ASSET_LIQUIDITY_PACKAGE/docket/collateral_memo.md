# Collateral Memorandum - Overmanifold Chat/IP Package

## Executive summary

This docket packages the Overmanifold conversation artifacts and uploaded files as a provenance-backed intellectual-property bundle. The package is designed to be reviewed as a collateral schedule, an appraisable asset register, and a liquidity-surface narrative.

The core asset is a coherent protocol thesis: **liquidity is reachable verified state**, and each Overmanifold endpoint can function as a bidirectional liquidity-capability surface.

The collateral pool consists of:

- Overmanifold protocol reports and PDFs.
- Native `.llm` prompt/provenance ledgers.
- UI screenshots proving product direction and visual identity.
- Chat/R&D archive zip.
- Text consolidation of tokenomics, endpoint mesh, LLM-SMS-OS, inverse mining, and provenance economics.
- Hash manifest and Merkle root for integrity.

## Collateral class

For financing discussion, this should be described as:

**Provenance-backed protocol intellectual property, software/documentation copyrights, trade-secret know-how, research corpus, semantic-ledger format, and prototype design materials.**

It should not be described as a guaranteed cash-equivalent asset.

## Asset / collateral / liquidity split

### As collateral
The package can support a secured loan, founder advance, grant, convertible note, or private underwriting conversation only if a counterparty accepts the IP and provenance record as collateral evidence. A normal bank is unlikely to treat raw files as loan collateral.

### As asset
The package is an R&D/IP asset bundle: it contains specifications, formal reports, technical architecture, prompt-chain ledgers, UI evidence, and source corpus.

### As liquidity
The package is liquid only after it is transformed into channels that others can transact against: license, repo, demo, whitepaper, grant application, tokenized provenance receipt, consulting deliverable, or revenue contract.
